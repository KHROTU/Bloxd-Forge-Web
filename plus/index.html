General Questions

Q: Why are there so few features?
A: There are currently only two active developers, plus both of us have school/work to do.
Q: Why isn't it open sourced?
A: The code behind the application contains sensitive information, so sadly we can't share it.
Q: What platforms does Bloxd Forge support?
A: Currently, Bloxd Forge is only available for Windows, but we are working on support for macOS and Linux systems.
Q: Is Bloxd Forge free to use?
A: Yes, Bloxd Forge is completely free, though we appreciate any support through our donation links.

Technical Issues

Q: My computer/virus scanning website says it's a virus. What should I do?
A: We try to use safe methods, but if we were to become a "trusted publisher", we would need to pay a hefty bill to Microsoft(around $500/yr), which is not happening anytime soon.
Q: Why isn't the AI chat working?
A: Forge Chat utilizes free Llama models, which may experience significant rate limitations. Additionally, Forge Chat Plus is not supported in the European Union.
Q: How do I report a bug?
A: You can report bugs through our Discord server or by creating an issue in our Github repository. Please include detailed information about the issue.
Q: Why am I experiencing crashes?
A: Crashes can occur due to insufficient system resources, conflicts with other software, or bugs in the application. Try updating to the latest version and checking your system requirements.

Software Problems

Q: The application is running slowly. How can I improve performance?
A: Try clearing the cache, ensuring your system meets minimum requirements, and closing unnecessary background applications.
Q: The application won't start. What should I check?
A: Verify you have the latest version installed, check if your antivirus is blocking it, and ensure all dependencies are installed correctly.
Q: How do I update Bloxd Forge?
A: You can check for updates in the settings menu, and download the newest version from the Github repository if nessecary
Q: Can I use Bloxd Forge offline?
A: Some features work offline, but most core functionality like texture pack downloading or AI chat requires an internet connection.

Support Us

Q: Can I support the developers?
A: Yes! Feel free to join our Discord and buy me a cup of coffee or some snacks.
Q: How can I contribute to the project?
A: You can contribute by reporting bugs, suggesting features, or helping with documentation through our GitHub repository or Discord server.

Account & Settings

Q: How do I change my settings?
A: You can access settings through the menu in the top-right corner of the application.

Troubleshooting

Q: What should I do if the application freezes?
A: Try restarting the application. If the issue persists, check for updates or reinstall the application.
Q: Why are some features not working?
A: Some features may be disabled due to compatibility issues or server limitations. Check our status page for updates.
