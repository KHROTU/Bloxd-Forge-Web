Starter guide

Here we will cover the basics of BloxdForge.

Note: World Forge will not be covered here as it is detailed in another guide.

Getting Started with BloxdForge

This guide will walk you through the essential steps of using BloxdForge, from installation to basic features.

1: Display Adjustment

Use Ctrl + to zoom in or Ctrl - to zoom out to optimize BloxdForge's display on your screen.

2: Menu Settings

The menu contains various self-explanatory options. The resolution setting, when set to 2, reduces game resolution to enhance performance on lower-end systems.

3: Tpack Management

Click the folder icon in the top-right corner to upload your tpack folder. Once pinned, access it anytime through the menu for easy drag-and-drop functionality.

4: Access Keys

Premium features like the Tpack editor require Access Keys. Obtain these keys for free by viewing advertisements in the Key section.

Creator Tab

The Texture Hub provides tools for creating and modifying Tpacks.

1: Accessing the Editor

Access the Tpack editor by entering your key obtained from the 'Key' tab.

2: Interface Navigation

After key validation, a sidebar will display various editing tools. Each option opens a specific functionality window.

3: Beginning Your Project

The primary editor is optimized for bloxd.io. Download textures from the GitHub repository and import them to start editing.

Important: Save your work regularly to prevent data loss.
