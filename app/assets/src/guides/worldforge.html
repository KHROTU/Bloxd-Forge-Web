World Forge guide

Here we will cover World Forge.

Using World Forge

World Forge is a powerful tool that helps you manage and edit your Bloxd.io world. Here's how to use its features effectively.

1. Left Sidebar - File Management

The left sidebar contains two main features:
- "Add Files": Upload .bloxdschem files to save your builds
- "Write Text": Create and save notes about your builds

2. World Edit Commands

In the right sidebar, you'll find World Edit commands:
- //pos1 and //pos2: Select an area
- //copy: Copy the selected area
- //paste: Paste your copied selection
- //schematic load: Load a saved build
- //schematic save: Save your current selection

3. Sign Creator

Create signs easily with the Sign Creator:
- Choose between "Press to Buy" or "Press to Get" signs
- Search for items from the complete item list
- Set amounts and costs for shop signs
- Generate sign text automatically

Tip: When using World Edit commands, they are automatically copied to your clipboard - just paste them in chat!

Managing Your Builds

Learn how to effectively save and manage your builds in World Forge.

1. Saving Builds


1. Select your build using //pos1 and //pos2
2. Use //copy to copy the selection
3. Use //schematic save to save it
4. Your build will be saved as a .bloxdschem file

2. Loading Builds


1. Upload your .bloxdschem file using "Add Files"
2. Use //schematic load to load it
3. Use //paste to place your build

Important: Always keep backups of your important builds by downloading the .bloxdschem files.

Tips and Tricks

Advanced tips for using World Forge effectively.

1. Organizing Builds

Use the text editor to keep notes about your builds, including coordinates and descriptions.

2. Quick Access

Keep frequently used builds in the left sidebar for quick access during building sessions.

3. Sign Management

Create templates for common signs to save time when setting up shops or information boards.

Remember: The World Edit menu can be collapsed to save space when not in use.
