Using PTC Generator

PTC Generator is a visual programming tool that helps you create Programmable Text Commands for Bloxd.io. Here's how to use its features effectively.

1. Block Categories

The left sidebar contains different categories of blocks:
- Input: Text input blocks
- Player Health: Health and shield management
- Combat: Combat-related commands
- Player Info: Player position and status
- Block Operations: Block manipulation
- Game UI: Interface commands
- Inventory: Inventory management

2. Block Programming

To create a script:
1. Drag blocks from categories
2. Connect blocks together
3. Fill in required values
4. Click "Generate Script" to create PTC code

Tip: Use the search bar to quickly find specific blocks you need.

Block Management

Learn how to effectively organize and use blocks.

1. Favorites System


- Mark frequently used blocks as favorites
- Access favorites from the Favorites category
- Customize your workspace for efficiency

2. Block Organization


- Use the search function to find blocks
- Read block descriptions in tooltips
- Organize blocks logically in your workspace

Important: Generated scripts are automatically formatted into 96-character blocks for game compatibility.
